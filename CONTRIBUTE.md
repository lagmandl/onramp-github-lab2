# Contributing to [Project Name]

Thank you for your interest in contributing! Here's how to get started.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/project-name.git`
3. Create a new branch: `git checkout -b feature/your-feature-name`

## Making Changes

- Follow the existing code style and conventions
- Write clear, descriptive commit messages
- Add or update tests as needed
- Update documentation if applicable

## Submitting a Pull Request

1. Push your changes to your fork
2. Open a pull request against the `main` branch
3. Describe what your changes do and why
4. Wait for a review — we aim to respond within [X days]

## Reporting Issues

- Search existing issues before opening a new one
- Include steps to reproduce, expected behavior, and actual behavior
- Attach relevant logs or screenshots if helpful

## Code of Conduct

Be respectful and constructive. See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) for details.

## Questions?
